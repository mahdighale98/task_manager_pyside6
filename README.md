# Task Manager (PySide6 + SQLite)

This is a simple Task Manager desktop application built with Python, PySide6, and SQLite.

---

## Status

This project is currently in development.

---

## Planned Features

- Add new tasks
- View all tasks
- Edit tasks
- Delete tasks
- Set task status (Done / Not Done)
- Set due date for tasks
- Save data using SQLite database

---

## Tech Stack

- Python
- PySide6 (Qt)
- SQLite (Database)
